# 概要

Angular で select/optionにngValueを使用するサンプルです。

# 実行環境

* Node.js 10.x
* Yarn 1.12.x

# 使用ライブラリ

* TypeScript 3.2.x
* Angular 7.2.x

# 動作確認

## 1. サンプルのダウンロード

```
git clone git@github.com:yasu-s/ng-combobox-sample.git
```

## 2. パッケージインストール  

```
cd ng-combobox-sample
yarn
```

## 3. サンプルの起動  

```
yarn start
```
